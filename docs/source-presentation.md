## What was extracted for the first documentation version

The first MkDocs version uses the presentation as the source for:

- project objectives
- imagery and publication workflows
- grassland and crop type tags
- event and status tags
- intensity and certainty classes
- the compact observation notation

## Editorial treatment

The slides contain draft-era wording, examples, and some spelling variations. This documentation keeps the underlying semantics while normalising formatting and turning presentation bullets into stable reference pages.

Future revisions should treat the presentation as historical source material and evolve the markdown pages directly under version control.
